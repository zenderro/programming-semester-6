Установка Qt

Есть несколько вариантов:

  1. Установка пакета в Ubuntu: sudo apt-get install qt5-default
  2. Установка старой версии через офлайн-установщик: https://www.qt.io/offline-installers
  3. Установка через онлайн-установщик (нужна регистрация): https://www.qt.io/download-qt-installer

При установке провьте, что отмечен Qt Creator, он может открыть файл проекта basic_graph.pro

Qt

Проще компилировать в терминале. Вместо Makefile используется файл проекта:

HEADERS       = window.h
SOURCES       = main.cpp \
              window.cpp
QT += widgets

Сборка (в каталоге с программой):

qmake        # qmake-qt5 в дисплейных классах
             # Для отладки: qmake CONFIG+=debug
make

// Функция-член
void Window::change_func ()
{
  // есть доступ к членам класса
  func_id = (func_id + 1) % 2;

  // Warning: если определить локальную переменную
  // с именем, совпадающим с именем члена класса
  // использоваться будет локальная переменная
  double func_id = -1.; // не делайте так

  func_id == -1.;

  // доступ к члену класса в этом случае:
  this->func_id
}

Рисование в QPainter I

https://doc.qt.io/qt-5/qpainter.html

Для перерисовки окна после обновления параметров вызывайте update().

Всё рисование выполняется в Window::paintEvent.

painter.setPen("black"); // установить цвет

Полный список цветов: https://doc.qt.io/qt-5/qcolor.html#setNamedColor

// линия от (x1, y1) до (x2, y2)
// в текущей системе координат
painter.drawLine(QPointF(x1, y1), QPointF(x2, y2));

Для рисования можно использовать количество точек width() или width()/2, меньше точек — хуже картинка.

Рисование в QPainter II

// сохранить систему координат
painter.save();

// восстановить сохранённую систему
painter.restore();
